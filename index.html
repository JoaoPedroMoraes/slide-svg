Desenho Vetorial

É o uso de primitivas geométricas como pontos, linhas, curvas e formas ou polígonos - todos os quais são baseados em expressões matemáticas - para representar imagens

Diferente de gráficos Raster, que usam pixels para formar a imagem

Adobe Illustrator lançado em 1985

O que é o SVG?

Scalable Vectorial Graphics

É arquivo XML que contem tags especificas para gerar uma imagem vetorial

Padrão criado pelo W3C em 2001

Atualmente na versão 1.1 (2ª edição)

Versão 2 esta em desenvolvimento

Compativel com todos os navegadores atuais

Beneficios

Comparação do tamanho dos arquivos por resolução

  • 300 x 300: SVG: 140KB / PNG: 101KB
  • 700 x 700: SVG: 140KB / PNG: 292KB
  • 1.500 x 1.500: SVG: 140KB / PNG: 1MB

Colocando SVG no HTML

Como Imagem

<img src="logo.svg" alt="Simples Ideias">

Inline SVG: Direto no corpo do html

<body>
    <svg>
    </svg>
</body>

Como Background-image: Por meio do CSS

.element {background-image: url(/image/image.svg);}
